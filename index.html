Web App Twitter
 

Automatic


Using Fourier transforms or some crap like that (actually Canny edge detection), we're able to use footage of Toomer's Corner to automatically.. wait... autoMAGICally detect whenever Toomer's Corner is rolled. Sure you could just guess by who wins a football game but hey, what fun is that? Also here is a picture of a camera.


Authentic


Our code is fed with 100% free-range text editors, made with love in good old Auburn, Alabama by four math PhDs. Or one math PhD and three PhD candidates. Our image detection is based on the Haskell functional programming language, while our frontend is an mix of Ruby and the completely unrecommended TPBaaS (Twitter-Powered-Backend-as-a-Service) programming pattern. And a little Bash scripting to glue it all together.
